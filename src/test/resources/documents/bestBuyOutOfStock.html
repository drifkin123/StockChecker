Description

The GeForce RTX 3080 Ti delivers the ultra performance that gamers crave, powered by AmpereNVIDIAs 2nd gen RTX architecture. Its built with enhanced RT Cores and Tensor Cores, new streaming multiprocessors, and superfast G6X memory for an amazing gaming experience.

Features

Graphics Processing Unit (GPU)/Clock Speed

Powered by the NVIDIA GeForce RTX 3080 Ti graphics processing unit (GPU). With a 1665 MHz boost clock speed to help meet the needs of demanding games.

On-board Memory

12 GB GDDR6X (384-bit)

Interface/Backward Compatibility

PCI Express 4.0 and earlier PCI Express 3.0 offer compatibility with a range of systems.

Filters

Antialiasing and anisotropic filtering deliver striking graphics with incredible realism.

API (Application Programming Interface) Support

DirectX 12 Ultimate, Vulkan RT, OpenGL 4.6 support

Audio support

HDMI and Display Port outputs enable flexible connectivity

NVIDIA Ampere Architecture

The all-new NVIDIA Ampere architecture features new 2nd generation Ray Tracing Cores and 3rd generation Tensor Cores with greater throughput. The NVIDIA Ampere streaming multiprocessors are the building blocks for extremely fast and efficient GPUs for gamers and creators.

NVIDIA Ampere Streaming Multiprocessors

All-new Ampere SM brings 2X the FP32 throughput and improved power efficiency.

2nd Generation RTCores

Experience 2X the throughput of 1st gen RT Cores, plus concurrent RT and shading for a whole new level of ray tracing performance.

3rd Generation Tensor Cores

Get up to 2X the throughput with structural sparsity and advanced AI algorithms such as DLSS. These cores deliver a massive boost in game performance and all-new AI capabilities.

What's Included

  • GeForce RTX 3080 Ti Graphics Card
  • NVIDIA Dual 8-pin power adapter
